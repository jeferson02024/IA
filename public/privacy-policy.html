Política de Privacidade - Nexia

Última atualização: 10 de março de 2026

O aplicativo Nexia foi desenvolvido para oferecer um assistente de inteligência artificial acessível e fácil de usar. Esta política descreve como tratamos as informações dos usuários.

1. Coleta de Dados

O Nexia não coleta dados pessoais diretamente. O aplicativo funciona como um WebView que carrega o site ia-2-uvqg.onrender.com. Quaisquer dados inseridos pelo usuário são processados pelo serviço de IA e não são armazenados pelo aplicativo.

2. Uso de Dados

As mensagens enviadas pelo usuário são utilizadas exclusivamente para gerar respostas da inteligência artificial. Não compartilhamos dados com terceiros para fins publicitários.

3. Cookies

O aplicativo pode utilizar cookies do navegador para manter sessões ativas. Esses cookies são gerenciados pelo próprio WebView do dispositivo.

4. Segurança

Nos preocupamos com a segurança dos dados dos usuários. Todas as comunicações são realizadas via HTTPS.

5. Menores de Idade

O Nexia não é direcionado a menores de 13 anos. Não coletamos intencionalmente dados de crianças.

6. Alterações nesta Política

Esta política pode ser atualizada periodicamente. Recomendamos que os usuários a revisem regularmente.
